Install Python:
1. Browse: python.org
2. click on latest Python version
3. On Windows Checkbox: Add Python to Path

Install Course Environment:
python -m venv d:\PythonCourse
cd PythonCourse\Scripts\
activate.bat

Install packages:
pip install numpy pandas jupyterlab

Install environment kernel:
python -m ipykernel install --user --name PythonCourse --display-name "PythonCourse"
